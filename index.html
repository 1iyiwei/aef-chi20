Autocomplete Element Fields

Chen-Yuan Hsu, Li-Yi Wei, Lihua You, Jian Jun Zhang

CHI 2020

Summary

A user interface for interactively brushing elements over 2D planes, 3D surfaces, and 3D volumes with partially or fully specified orientation fields assisted by automatic synthesis.

This is a new version of the method described in a SIGGRAPH Asia 2018 technical brief, using a procedural instead of data-driven method to place elements, resulting in a faster and easier to use system (requiring only individual element shapes but not their overall distributions as input).
